Managing Research Software Projects

About the Workshop

This workshop introduces the ideas and tools you need to manage a team of up to a dozen people working together to build research software. Each session ends with something you can take home and use the following week. Sessions 1-9 form a single day; Sessions 10-15 are additional half-day modules that can be added individually.
